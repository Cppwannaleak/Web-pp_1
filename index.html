기술 스택입니다.

프론트앤드는 Vue3, Vite, Typescript를 사용한 개발을 선호합니다. 스타일 구성은 Bootstrap template 사용이 편합니다.

임베디드보드는 STM32,ESP 칩을 주로 사용합니다. PCB 설계는 하지 않고 플랫폼 보드만 사용해본 경험이 있습니다.

백엔드는 Firebase나 Lamda같은 서버리스 솔루션 위주로 개발합니다. 보드에 올리는 엣지 컴퓨팅용 경량 웹 서버로 Flask+sqlite 사용 경험 있습니다.

데이터 처리는 Opencv, tensorflow, scikit-learn 사용 경험 있습니다.

Desktop App은 C#[WPF] 사용 선호합니만, QT나 MFC도 가능은 합니다.

개발환경은 Arch Linux의 텍스트 에디터 NeoVim을 사용합니다.

서브 시스템으로는 Toolchain, Git, HDFS, Docker 사용가능합니다.
